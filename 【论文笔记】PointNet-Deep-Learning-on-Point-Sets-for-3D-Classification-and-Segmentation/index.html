【论文笔记】PointNet-Deep Learning on Point Sets for 3D Classification and Segmentation

发表于 2019-01-31 | 更新于 2019-02-01 | 分类于 论文笔记 | 阅读次数：

在这篇论文之前的大多数处理点云的方法都是将空间划分成很多的voxel grids。这么做尽管取得了不错的效果，但是也造成了信息的损失。而作者提出的这个PointNet充分尊重了点云的无序性，同时也高效地完成了图像分类和分割任务。

整个网络的结构示意图如下所示：

作者强调在这个网络中有三个关键模块：一个最大池化层（max pooling layer），一个局部和全局信息结合器（local and global information combination structure），和两个调序网络（alignment network）。池化层的目的是充当一个对称函数。因为对于无序的点云数据，有三种方法可以处理：1）排序；2）全排列；3）对称函数。排序看似简单，但在高维度空间里并不存在一种稳定可靠的排序方法。全排列随着数据量增大，耗时越发不可接受。所以对称函数是一个可行的方法，这里采用max pooling。结合器是为了结合局部和全局特征，是。调序网络则是用一个叫T-net的小型网络来学习一个转换矩阵，分别在输入数据和第一次提取特征处对数据做一个转换。这个T-net包含了特征提取、池化和全连接，目的是希望点云数据在经历刚性变换之后我们学习到的特征表示能够保持不变。
